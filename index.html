Atividade tags HTML semântica

Uso de negrito

Posso fazer o negrito usando a tag B, mas ela não tem semântica

É melhor usar a tag STRONG, pois ela é semântica

Uso do Itálico/ênfase

Posso usar a tag i para fazer um ITÁLICO, mas não terá valor semântico

É melhor usar a tag EM, pois ela é semântica e passa o significado de ênfase naquele trecho de texto

Texto marcado

É possível fazer um texto marcado com a tag MARK
